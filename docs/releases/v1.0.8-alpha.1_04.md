# Alpha 1.0.8_04

Updater handoff and runtime recovery hotfix release for the Alpha 1.0.8 line.

## Included

- Fixes web updater requests getting stuck in `queued` by validating updater runtime health before queueing and failing stale queued handoffs after the pickup grace window.
- Replaces the fragile single request file handoff with queued request files while keeping legacy request-file compatibility during rollout.
- Makes updater fixes self-propagating by refreshing `despatch-updater.path` and `despatch-updater.service` from release payloads during apply.
- Improves Admin updater diagnostics for inactive path units, missing service/runtime wiring, and broken worker executable resolution.
