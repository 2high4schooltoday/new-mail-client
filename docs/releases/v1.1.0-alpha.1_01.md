# Alpha 1.1.0_01

UI artifact and alignment cleanup release for the Alpha 1.1 line.

## Included

- Cleans up the app shell and Mail header hierarchy so top-level navigation, utility controls, and workspace titles align more consistently without duplicated emphasis.
- Reworks the Mail toolbar and account-health disclosure so health opens as an anchored popover instead of pushing the toolbar out of alignment, while tightening reader controls and message-row layout.
- Fixes visual artifacts from browser-native focus, select, date, and radio rendering by normalizing controls to the machine-dark UI system.
- Simplifies browse/detail screens across Settings, Senders, Contacts, Rules, and Scripts with calmer empty states, cleaner list rows, shorter metadata, and better-aligned forms.
