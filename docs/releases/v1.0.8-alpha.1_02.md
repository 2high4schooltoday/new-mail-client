# Alpha 1.0.8_02

Mail reliability and updater/runtime hotfix release for the Alpha 1.0.8 line.

## Included

- Fixes HTML reader rendering on dark theme (readable iframe foreground/background instead of black-on-transparent output).
- Hardens message preview extraction with MIME-aware snippet parsing (`text/plain` first, `text/html` fallback) and removes noisy `(no preview)` placeholders.
- Ships cross-mailbox conversation threading (`scope=conversation`) with Inbox/Sent/Archive scanning, additive mailbox metadata, and bounded scan caps.
- Polishes mail hierarchy ergonomics (grouped toolbar actions, softened pane chrome, sender-first list scanning, improved reader/thread context text).
- Includes updater/runtime contract hardening already in this patch set to prevent updater path/permission drift regressions.
