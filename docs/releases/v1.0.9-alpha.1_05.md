# Alpha 1.0.9_05

Updater lock-permission hotfix release for the Alpha 1.0.9 line.

## Included

- Fixes web updater failures caused by unprivileged status/apply code trying to `stat` the root-only updater lock path.
- Keeps updater lock directory permissions intact while making web-side queue and status checks permission-safe.
- Adds regression coverage for unreadable updater lock-directory scenarios so this failure does not return.
