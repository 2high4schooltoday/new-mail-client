# Alpha 1.0.14_01

Indexed mail draft/send hotfix release for the Alpha 1.0.14 line.

## Included

- Fixes indexed-account mail so users can open Mail, save drafts, reopen drafts, and send replies even when the session mailbox secret is still locked.
- Adds account-scoped mailbox and indexed attachment APIs so the indexed workspace no longer falls back to the session-only mail path for core reader and compose flows.
- Preserves session-mail compatibility while tightening draft ownership checks and adding regression coverage for indexed draft/send reliability.
