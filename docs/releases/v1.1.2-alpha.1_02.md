# Alpha 1.1.2_02

Typography and mail-reader hotfix for the Alpha 1.1.2 line.

## Included

- Enforces `IBM Plex Mono` across the app UI, helper surfaces, compose HTML preview, and generated document views by removing the remaining alternate font declarations.
- Adds a UI contract guard that now fails the release check if non-Plex font declarations reappear in the web source files.
- Keeps received HTML messages faithful to their embedded typography by allowing the HTML reader to honor sender-defined `font-family` rules instead of force-overriding them.
- Extends the mail reader regression to verify sender-defined HTML fonts still render correctly while preserving iframe sizing behavior.
