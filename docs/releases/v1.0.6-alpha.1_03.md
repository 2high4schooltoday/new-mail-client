# Alpha 1.0.6_03

Installer automation hardening for TLS and PAM/IMAP defaults to reduce manual post-install fixes.

## Included

- Auto-detects Let’s Encrypt certificate/key pairs from `/etc/letsencrypt/live` for reverse proxy TLS setup.
- Prefills and auto-recovers TLS cert/key paths during install when valid LE files are present.
- Enables safer PAM IMAP loopback defaults by auto-setting `IMAP_INSECURE_SKIP_VERIFY=true` when TLS/STARTTLS uses loopback.
- Updates `despatch.py` install flows (TUI/plain/legacy) to auto-seed detected TLS settings.
