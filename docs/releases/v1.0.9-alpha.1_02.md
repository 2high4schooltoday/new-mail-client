# Alpha 1.0.9_02

TUI installer clarity and license-gate hotfix release for the Alpha 1.0.9 line.

## Included

- Fixes remaining TUI rendering artifacts by removing rail/content banding and stabilizing the installer shell layout.
- Simplifies the installer copy so setup screens use clearer, less technical language while keeping the existing guided flow.
- Adds an in-installer Software License step that must be accepted before install or upgrade can continue.
- Improves installer progress and completion screens so detailed technical output stays in the log drawer instead of dominating the main UI.
