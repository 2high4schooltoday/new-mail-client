# Alpha 1.0.9_03

TUI compatibility and license-loading hotfix release for the Alpha 1.0.9 line.

## Included

- Fixes remaining TUI shell artifacts by removing artifact-prone rail/card chrome from the live curses renderer.
- Makes `DESPATCH_TUI_ASCII=1` a real ASCII-safe fallback instead of still relying on terminal pseudo-graphics.
- Fixes the installer license step to fetch `LICENSE.md` from the GitHub repository at runtime, with local fallback when needed.
- Adds regression coverage for GitHub license loading and the updated TUI wrapping/rendering behavior.
