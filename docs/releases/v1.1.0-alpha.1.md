# Alpha 1.1.0

Content-first mail UI cleanup and conversation reliability release for the Alpha 1 line.

## Included

- Reworks the main app flow so Mail and Contacts stay primary while Settings and Admin move into utility navigation, with cleaner headers, toast status feedback, and a more focused Mail toolbar.
- Cleans up Mail interaction details with a denser inspect menu, reliable in-app printing, browse-first Settings and Contacts detail views, and clearer sender/account wording in compose and Mail settings.
- Fixes conversation threading across full syncs, cross-mailbox replies, and historical indexed mail by processing sync batches oldest-first, repairing stored thread ancestry by message headers, and rerunning one-time thread repair on upgrade.
- Corrects raw message reply-header parsing so `In-Reply-To` and `References` no longer poison stored thread roots, with new regression coverage to keep split conversations from coming back.
