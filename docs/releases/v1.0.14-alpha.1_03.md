# Alpha 1.0.14_03

Mail session bootstrap and draft-save hotfix release for the Alpha 1.0.14 line.

## Included

- Auto-creates the indexed mail account for PAM-authenticated users from the verified mailbox login and stored session secret, so indexed mail no longer requires a separate manual mapping.
- Accepts the compose client's blank `scheduled_for` value on draft create, restoring draft save and send flows that were failing before mail send could start.
