# Alpha 1.0.13

Mail identity and signature release for the Alpha 1.0 line.

## Included

- Adds real sender identities with display names, reply-to, and per-identity signatures for both session mail and account-backed mail.
- Upgrades compose and drafts so signatures insert once, survive reopen, and switch cleanly when the selected sender changes.
- Fixes the updater service sandbox so privileged update runs can write their state under `ProtectSystem=strict`.
