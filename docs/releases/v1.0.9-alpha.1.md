# Alpha 1.0.9

Installer assistant release for the Alpha 1.0 line.

## Included

- Reworks the curses TUI into a guided installer assistant with welcome, review, progress, and completion flows.
- Adds richer terminal rendering with Unicode-first progress, step rail, and safe ASCII fallback via `DESPATCH_TUI_ASCII=1`.
- Keeps existing installer semantics and staged checks while improving layout hierarchy, spacing, and compact-terminal behavior.
