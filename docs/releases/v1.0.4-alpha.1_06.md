# Alpha 1.0.4_06

Passkey remediation + passwordless sign-in rollout.

## Included

- Adds passwordless passkey login endpoints (`/api/v1` and `/api/v2`) with challenge cookie binding, expiry handling, replay blocking, and required user verification.
- Adds passkey capability discovery (`GET /api/v1/public/auth/capabilities`) and stricter WebAuthn context checks (HTTPS/localhost, RP/origin mismatch errors).
- Adds passkey management UX/API: list + rename + delete credentials in Security settings, plus login-page passkey sign-in button.
- Adds persisted encrypted mail-secret bootstrap flow for passkey sessions (`mail_secret_required` + `/session/mail-secret/unlock`) so mailbox access can continue without password-form login.
- Includes migrations/config/env updates for passkey + mail-secret support and expands API tests for passkey login, challenge replay rejection, and unlock flow.
