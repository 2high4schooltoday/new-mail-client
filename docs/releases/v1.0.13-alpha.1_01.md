# Alpha 1.0.13_01

Updater deployment hotfix release for the Alpha 1.0.13 line.

## Included

- Fixes updater-deployed runtime tree permissions so `despatch.service` can read `web/`, `migrations/`, and `deploy/` after restart.
- Prevents update rollback loops caused by startup failing with `permission denied` on migration files.
- Adds updater regression coverage for deployed runtime permission normalization in both updater implementations.
