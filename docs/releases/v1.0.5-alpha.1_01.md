# Alpha 1.0.5_01

Updater reliability fix for Ubuntu deployments with read-only system unit paths.

## Included

- Fixes update failure when installing `mailclient-mailsec.service` hits `Read-only file system`.
- Allows update to continue when a valid existing mailsec unit is already known by systemd.
- Keeps strict failure behavior when mailsec is enabled but no usable unit exists.
- Applies parity fixes in both updater engines (Go fallback and Rust `update_worker`).
