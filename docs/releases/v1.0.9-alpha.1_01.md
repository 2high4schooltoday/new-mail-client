# Alpha 1.0.9_01

TUI stability and rendering hotfix release for the Alpha 1.0.9 line.

## Included

- Fixes TUI shell artifacts caused by divider collisions, overdraw in the rail, and noisy full-width field separators.
- Stabilizes installer form, review, progress, and completion layouts with cleaner wrapping, aligned value columns, and calmer footer actions.
- Softens focus and button styling while keeping the Despatch TUI identity and preserving UTF-8 and ASCII fallback rendering.
- Expands renderer preview tests so layout regressions are caught before release.
