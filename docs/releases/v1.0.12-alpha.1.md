# Alpha 1.0.12

Mailbox state and conversation reliability release for the Alpha 1.0 line.

## Included

- Makes mailbox state update immediately for read/unread, moves, archive/trash, and sent-copy placement.
- Adds persistent `Sent`, `Archive`, and `Trash` special-folder resolution so first-use actions create or bind the expected mailbox.
- Keeps the message list, unread counts, and reader thread rail refreshed in place without requiring manual folder switching.
