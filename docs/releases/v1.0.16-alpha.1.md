# Alpha 1.0.16

Advanced mail filtering and message inspection release for the Alpha 1.0 line.

## Included

- Replaces plain indexed mail search with an advanced filter bar for free-text, sender, recipient, subject, date range, unread, flagged, attachment, waiting, and per-account refinement, with saved-view compatibility.
- Fixes noisy indexed message previews so CSS, raw HTML, header fragments, and other machine noise no longer leak into the message list, including repair of already-stored bad snippets at read time.
- Adds reader inspect tools for both live and indexed mail: raw message source, full headers, print, and `.eml` export, backed by live and indexed RFC822 raw-message endpoints.
